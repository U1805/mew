Mew Bot SDK 是扩展 Mew 生态能力的核心工具。通过它，你可以构建自动化服务、集成第三方应用，或创建丰富的交互式体验。

本指南将引导你使用 SDK 构建两种核心类型的机器人，并提供一个完整的端到端测试流程。

## 两种核心机器人类型

在开始之前，你需要了解 Mew 中两种 Bot 的设计定位：

*   **Fetcher Bot (单向推送)**
    负责从外部数据源（如 RSS、API）抓取信息，并单向推送到 Mew 的指定频道中。它是一个信息聚合与分发的“搬运工”。

*   **Agent Bot (双向会话)**
    能够与用户进行实时交互，可以响应用户的指令、提供查询服务或进行多轮对话。它是一个可交互的“智能体”。
